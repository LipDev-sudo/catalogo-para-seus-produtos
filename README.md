# Catalogo Online Profissional

Vitrine digital para pequenos negocios que querem iniciar vendas online com baixo investimento. Modelo comercial pronto para uso.

## Preview

[Ver Demo](https://catalogo-para-seus-produtos.vercel.app/)

## Tecnologias

- React
- TypeScript
- Vite
- Tailwind CSS

## Como rodar

```bash
npm install
npm run dev
```

## Funcionalidades

- Listagem de produtos com grid responsivo
- Pagina individual de produto
- Carrinho visual
- Layout responsivo para todos os dispositivos
- Estrutura escalavel e facil de customizar
- Ideal para pequenos comercios e empreendedores

## Objetivo

Modelo comercial para clientes que desejam iniciar vendas online com baixo investimento. Pronto para personalizar e entregar.
